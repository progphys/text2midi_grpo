# Melody Focus Prompt Bank

Этот набор нужен для случаев, когда нам важно не столько строго задавать стиль, сколько стабильно получать:

- явную основную мелодию;
- верхнюю ведущую линию;
- фоновый support вместо конкурирующих лидов.

## Suggested Preset

В synthetic training теперь можно использовать:

```yaml
prompt:
  preset: melody_focus_bank
```
